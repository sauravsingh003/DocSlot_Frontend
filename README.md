# DocSlot Frontend

React frontend for the DocSlot online appointment system.

## Environment

Create a `.env` file in the frontend root:

```env
VITE_API_BASE_URL=https://docslot-backend-aotc.onrender.com
```

Set the same `VITE_API_BASE_URL` value in Vercel project environment variables.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

Production builds are generated in the `dist` folder.
